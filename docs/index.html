What is Procedural Generation?

Creating data using algorithms and computer-generated randomness.

Problems in this field

  1. High barrier of entry due to advanced prerequisite knowledge
  2. Expensive to implement and maintain
  3. Lack of advanced algorithms plugins in the marketplace

Our Solution

A procedural generation toolkit that:

  1. Offers a wide variety of algorithms
  2. Abstracts away complexity
  3. Can select from and configure multiple algorithms
  4. Wrapped up in an easy-to-use toolkit deployed Unity

Lindenmayer System

Rule-based grammar system originally made to model plant growth.

Our Difference

Rich GUI to abstract complexities when constructing L-system grammar trees
